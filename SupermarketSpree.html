  • I created this project very recently over January 2021 and researched the bone structure of skeletal meshes and how they are manipulated. This mechanic lead me to attach a static mesh to the hand of my character and this, along with an animation, created the illusion of my character picking up an object.



  • Within this project, I created an inventory system that allows the player to pick up any item in any order and this will be added to the inventory along with the number of different items they have collected. This can be toggled on and off by pressing the "I" key.



  • Alongside this, I toggled the visibility of the shopping cart on the third person character, creating and destroying the trolley actor so the player can accurately pick up the trolley and place it back down. The player can push/let go of the trolley by pressing the "T" key.
