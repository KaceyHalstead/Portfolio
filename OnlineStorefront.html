  • I created this online storefront October/November 2020 as part of a college assignment. During this project, I further developed my understanding of arrays and refined my knowledge of user interfaces, as stated below:



  • Within this project, I studied user interfaces further to achieve the most user-friendly layout for an online storefront, taking feedback from regular storefront users about current competitors to see where I could improve upon this.



  • I learnt more efficient ways of storing, obtaining and organising data which lead me to build an efficient shopping cart and wishlist system. This saved me a lot of time as I did not have to create individual interfaces for every game.



  • I made a fully functional log-in screen that allowed the user to enter a username and password (or create a new account if they don't have one) to log in. All accounts have separate credits and this is kept track of individually.
