  • I created this text-based Rock, Paper, Scissors game in Python. During GCSE Computer Science, I learnt Python and began to understand the basic fundamentals of programming. This was a fun project I created in my own time and within this, I specifcially looked into storing data externally in a text file.



  • This is how the game functions within the Python shell. The program determines whether the player wins, the computer wins, or whether it was a tie. The overall winner is the first to three wins.



  • This is the CSV file I used to store the number of overall wins by the player within one run of the game. When the game is closed, the PlayerWins value will reset and the player will have to beat the highscore for their score to be remembered.
