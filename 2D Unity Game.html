  • I created this 3-level 2D platformer as part of my Computer Science A Level course. This project required a lot of research and dedication as this was my first time creating a game in Unity, however, I was much more comfortable using Unity and programming in C# by the end of the assignment. This was a great learning experience and has inspired me to learn more about this game engine.



  • During this project, I learnt how to create user interfaces on Unity, which lead me to make a fully functiuonal Main Menu, HUD and pause menu. Also, using player prefs, I was able to implement a highscore system that calculated the player's average time and place it on the Main Menu screen.



  • This is an example of my "CloudScript" that makes the cloud disappear after 5 seconds of the player standing on it, and reappear 3 seconds later. This was to add an extra challenge to the game.
